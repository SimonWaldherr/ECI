ECI Message Definitions¶

ECI_NOOP_CC¶
Description:No-Op Command
Command Verification:
 HK command counter increments and event# ECI_NOOP_INF_EID issued
Error Condition:
 None
ECI_RESET_HK_COUNTER_CC¶
Description:HK Counters Reset
Command Verification:
 HK command, error, and Msg counters reset to zero, event# ECI_RESET_HK_CTR_INF_EID issued
Error Condition:
 None
