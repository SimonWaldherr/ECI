ECI Table Interface¶

ECI_PARAM_TBL_MAX_NAME_LEN¶
Maximum parameter table length. If this variable isn’t defined
it will be set to CFE_TBL_MAX_NAME_LENGTH.
ECI_TBL_FILEDEF(StructName, ObjName, TblName, Desc, Filename)¶
Function to create table files via the elf2cfetbl tool. This should be called
at the end of the parameter table header file to be later compiled with elf2cfetbl. This macro uses expansion to resolve it’s names so there should be no commented strings as parameters.
Parameters:
  • StructName – Data structure type to be used as the table image format
  • ObjName – Name of the object to be placed in the parameter table
  • TblName – Name of the table registered with ECI_Tbl_t.tblname
  • Desc – Quick description of the table
  • Filename – What to name the file
Returns:

Static declaration of CFE_TBL_FileDef_t to be used with elf2cfetbl

Example

ECI_TBL_FILEDEF(CFE_TBL_FileDef, tle_lines_tbl, OP_APP.OP_TLE_Tbl,
    OP app TLE table, tle_tbl.tbl )
